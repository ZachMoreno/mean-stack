Custom MEAN Stack
============

## Dependencies
 - Back Ends
   - Node.js
     - Express.js
     - MongoDB
       - Mongoose
   - Firebase
     - Angular-Fire
       - Route-Security
       - Simple-Login-Tools

 - Front End
   - AngularJS
     - Angular-Route
   - Foundation Framework
     - jQuery
     - FastClick
     - Modernizr

## Tools
 - NPM
 - Bower
 - Nodemon

## Back End Setups
### Node.js
 1. Download & install Node.js
 2. Download & install MongoDB
 3. `sudo npm install -g nodemon`
 4. `git clone git@github.com:ZachMoreno/mean-stack.git`
 5. `cd mean-stack/back-end`
 6. `npm install`
 7. `cd ../front-end`
 8. `bower init`
 9. Answer `Y` to add existing dependencies to project?

### Firebase
 1. Create Firebase account
 2. Create a new Firebase instance for your app
 3. ...
